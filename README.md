# 息壤 Xi-Rang

> 息壤：中国神话中能自己生长、永不耗尽的土壤。

状态机驱动的自动化资产配置系统。稳定优先，抗通胀，防止人为犯错。

## 它是什么

一个跑在服务器上的 Python 程序，每天收盘后自动拉取 ETF 行情，通过风控状态机决定是否调仓，把结果记录到 SQLite，有动作时推送通知。入金可以自动化；出金只允许发起请求，后续必须人工审批、人工处理。

## 它不是什么

- 不是交易机器人（不自动下单，只生成指令）
- 不是预测系统（不判断涨跌，只做再平衡）
- 不是高频策略（一年调仓约 50 次）

## 核心策略

四资产等权永久组合 + 双层风控断路器。

```
正常模式:   SPY 25% | TLT 25% | GLD 25% | SHV 25%
常规保护:   SPY 10% | TLT 20% | GLD 20% | SHV 50%   ← 回撤 -12% 触发
紧急避险:   SPY  3% | TLT  7% | GLD 15% | SHV 75%   ← 回撤 -14% 触发
```

## 回测成绩（2005-2025，21年）

```
年化收益(CAGR):  7.34%
最大回撤(MDD):  -14.81%
夏普比率:        0.77
调仓次数:        51 次（平均每年 2.4 次）
保护模式:        278 天（占比 5.3%）
```

三项成功标准全部通过：CAGR 跑赢通胀+2%，MDD 控制在 -15% 以内，夏普 > 0.5。

## 7.34% 到底意味着什么

7.34% 的年化收益率看起来不"性感"。但复利的威力不在于速度，在于时间。

### 美国市场（2005-2025，21 年）

标的：SPY（标普500）+ TLT（长期国债）+ GLD（黄金）+ SHV（短期国债）

**一次性投入 10 万美元：**

| 方式 | 年化 | 21 年后 | 纯利润 |
|------|------|---------|--------|
| 银行定期（2.5%） | 2.5% | **$16.8 万** | $6.8 万 |
| 息壤 | 7.34% | **$44.2 万** | $34.2 万 |

**初始 10 万 + 每年定存 5 万美元（总投入 $115 万）：**

| 方式 | 年化 | 21 年后 | 纯利润 | 息壤多赚 |
|------|------|---------|--------|---------|
| 银行定期（2.5%） | 2.5% | **$156 万** | $41 万 | — |
| 息壤 | 7.34% | **$295 万** | $180 万 | **+$139 万** |

295 万美元按 7.34% 年化，每年被动收益约 21.6 万——远超每年存入的 5 万。财富开始自我繁殖。

### 中国市场（2012-2025，14 年）

标的：沪深300 ETF + 国债 ETF + 黄金 ETF + 货币 ETF

```
年化收益(CAGR):  5.39%    最大回撤(MDD):  -12.97%
夏普比率:        0.53     三项成功标准全部通过
```

**一次性投入 10 万人民币：**

| 方式 | 年化 | 14 年后 | 纯利润 |
|------|------|---------|--------|
| 银行定期（2.5%） | 2.5% | **¥14.1 万** | ¥4.1 万 |
| 息壤 | 5.39% | **¥20.5 万** | ¥10.5 万 |

**初始 10 万 + 每年定存 5 万人民币（总投入 ¥80 万）：**

| 方式 | 年化 | 14 年后 | 纯利润 | 息壤多赚 |
|------|------|---------|--------|---------|
| 银行定期（2.5%） | 2.5% | **¥93 万** | ¥13 万 | — |
| 息壤 | 5.39% | **¥109 万** | ¥29 万 | **+¥16 万** |

中国市场收益低于美股约 2 个百分点（A 股长期回报不如美股），但回撤反而更小（-12.97% vs -14.81%），风控引擎在 A 股暴涨暴跌的环境下更有效。

### 两个市场的对比

| 指标 | 美股（21年） | 中国（14年） |
|------|------------|------------|
| CAGR | 7.34% | 5.39% |
| MDD | -14.81% | -12.97% |
| 夏普比率 | 0.77 | 0.53 |
| 保护模式触发 | 11 次 | 6 次 |
| 保护模式占比 | 5.3% | 3.7% |

策略逻辑不依赖某个特定市场，它依赖的是"股票、债券、黄金、现金四类资产的长期负相关性"。这个特性在中美两个市场都成立。

### 混血配置：用国内券商账户买标普500 ETF

既然 A 股成长性不足，能不能在国内券商直接买标普500 ETF（513500.SS），搭配国内的国债、黄金和货币基金？

回测结果（2013-2025，13 年）：

| 指标 | 混血配置 | 纯中国配置 | 差异 |
|------|---------|-----------|------|
| CAGR | 5.98% | 5.79% | +0.18% |
| MDD | -15.97% | -12.97% | -3.00% |
| 夏普比率 | 0.62 | 0.59 | +0.03 |
| 调仓次数 | 15 | 31 | -16 |
| 保护模式触发 | 1 次 | 6 次 | -5 |

**定投场景（初始 ¥10 万 + 每年 ¥5 万，13 年，总投入 ¥70 万）：**

| 方式 | 终值 | 利润 |
|------|------|------|
| 混血配置 | **¥104.3 万** | ¥34.3 万 |
| 纯中国配置 | **¥103.0 万** | ¥33.0 万 |
| 银行定期（2.5%） | **¥82.4 万** | ¥12.4 万 |

混血配置的 CAGR 略高，调仓更少（15 次 vs 31 次），但 MDD 突破了 -15% 红线（-15.97%），原因是标普500 ETF 上市早期流动性差、溢价波动大。纯中国配置反而更稳，三项标准全部通过。

现实中混血配置还面临 QDII 额度限制、ETF 溢价率波动、汇率风险等问题。如果能接受这些，长期收益确实更高，但需要针对性调优风控参数。

### 欧洲市场：息壤的边界在哪里

我们也用同样的策略跑了欧洲市场（EZU 欧元区股票 + BWX 国际国债 + GLD 黄金 + SHV 现金，2005-2025）。

```
年化收益(CAGR):  3.77%    最大回撤(MDD):  -17.06%
夏普比率:        0.25     三项成功标准全部未通过
保护模式占比:    45.6%    调仓次数: 113 次
```

欧洲三项全挂：CAGR 没跑赢通胀、MDD 突破红线、夏普远低于 0.5。核心原因是欧元区股票过去 20 年极其疲软——2008 金融危机、2011 欧债危机、2022 俄乌战争 + 能源危机，结构性打击太多。系统 45% 的时间都在保护模式，说明欧洲市场的危机频率已经超出了策略的设计边界。

不过即便如此，定投 20 年（初始 $10 万 + 每年 $5 万）息壤欧洲版仍然比银行多赚了 $22 万。它不是完全没用，只是不够好。

### 全球配置：买下全人类的商业文明

既然欧洲不行，能不能用 VT（Vanguard 全球股票 ETF，覆盖 49 个国家）替代 SPY，彻底消除单一国家的国运风险？

回测结果（2007-2025，同一时间窗口对比）：

| 指标 | 全球配置 (VT) | 美股原版 (SPY) | 差异 |
|------|-------------|--------------|------|
| CAGR | 4.44% | 7.05% | -2.61% |
| MDD | -15.55% | -14.81% | -0.74% |
| 夏普比率 | 0.36 | 0.72 | -0.36 |
| 保护模式占比 | 20.0% | 5.0% | +15.0% |

**定投场景（初始 $10 万 + 每年 $5 万，18 年，总投入 $100 万）：**

| 方式 | 终值 | 利润 |
|------|------|------|
| 美股原版 | **$204.9 万** | $104.9 万 |
| 全球配置 | **$155.3 万** | $55.3 万 |
| 银行定期（2.5%） | **$127.5 万** | $27.5 万 |

全球配置在过去 17 年跑输美股原版，因为这恰好是美国一家独大的时代。VT 里约 60% 是美股，剩下 40% 的欧洲、日本、新兴市场拖了后腿。

但全球配置赌的不是过去，而是未来。如果下一个 20 年美国不再独霸，VT 会自动调整权重跟随新的赢家，而 SPY 不会。这是一个"保险"的选择，代价是牺牲一部分收益。

### 全球六配置总览

| 配置 | 时间跨度 | CAGR | MDD | 夏普 | 成功标准 | 适合场景 |
|------|---------|------|-----|------|---------|---------|
| 美股原版 | 21 年 | 7.34% | -14.81% | 0.77 | ✓ 全通过 | 有美股账户，信任美国国运 |
| 纯中国 | 14 年 | 5.39% | -12.97% | 0.53 | ✓ 全通过 | 只用 A 股账户 |
| 中美混血 | 13 年 | 5.98% | -15.97% | 0.62 | ✗ MDD 超标 | A 股买标普 ETF |
| 全球配置 | 19 年 | 4.44% | -15.55% | 0.36 | ✗ 未全通过 | 不赌任何单一国家 |
| 印度 | 19 年 | 3.32% | -18.46% | 0.20 | ✗ 全未通过 | 不推荐 |
| 欧洲 | 21 年 | 3.77% | -17.06% | 0.25 | ✗ 全未通过 | 不推荐独立运行 |

印度市场三项全挂，MDD -18.46% 是所有配置中最大的。印度股市波动率极高（暴涨暴跌、政策风险、外资进出剧烈），导致风控引擎 45% 的时间都在保护模式，错过了大量反弹。新兴市场的高波动性超出了息壤"稳定优先"策略的设计边界。

选择哪个配置，本质上是在回答一个问题：你相不相信美国会继续赢下去？如果相信，选美股原版；如果不确定，选全球配置买个保险；如果只能用 A 股账户，纯中国配置已经足够好。

### 为什么"不性感"才是对的

息壤不是用来暴富的。它是一面盾，不是一把剑。

- 单吊一只股票翻 10 倍的故事很性感，但 99% 的人死在了暴跌里
- 7.34% 不刺激，但它穿越了 2008 金融危机、2020 疫情熔断、2022 股债双杀，活了下来
- 最大回撤 -14.81%，意味着最坏的时候你亏不到 15%，晚上睡得着觉
- 真正的财富来自"本金 × 时间 × 纪律"，不是来自预测和赌博

> 息壤的价值不是让你一夜暴富，而是让你的每一分劳动收入都不被通胀吞噬，在 20 年后长成一片自我生长的土地。

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 准备数据（拉取 2005-2025 ETF + CPI 数据）
python data/data_loader.py

# 跑一次回测验证
python -m backtest.engine_backtest

# 启动 Web 服务（仪表盘 + API）
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
# 浏览器访问 http://localhost:8000

# 或仅运行每日调仓（无需 Web）
python -m runner.daily_runner

# 查看报告
python -m runner.report
```

Web 服务启动后，所有前端资源（Chart.js、CSS）均为本地文件，零外部 CDN 依赖，断网可用。

## 部署到服务器

息壤采用"本地拉数据 → GitHub 中转 → 服务器读取"的数据管道，解决国内服务器无法访问金融 API 的问题。

```
本地电脑 (yfinance) → CSV → git push → GitHub → 服务器 git pull → daily_runner
```

### 服务器端

```bash
git clone https://github.com/jihongxing/BreathofEarth.git /opt/xirang
cd /opt/xirang
pip3 install -r requirements.txt

# 首次运行
python3 -m runner.daily_runner --force

# 配置 cron（每天 06:30 先拉数据再运行，北京时间）
crontab -e
# 30 6 * * 2-6 cd /opt/xirang && git pull -q && /usr/bin/python3 -m runner.daily_runner >> /opt/xirang/logs/cron.log 2>&1
```

### 本地电脑（Windows）

每天自动拉取中美行情并推送到 GitHub：

```powershell
# PowerShell（管理员）运行一次即可
$action = New-ScheduledTaskAction -Execute "python" -Argument "D:\codeSpace\BreathofEarth\data\daily_fetch.py" -WorkingDirectory "D:\codeSpace\BreathofEarth"
$trigger1 = New-ScheduledTaskTrigger -Daily -At 11:05AM
$trigger2 = New-ScheduledTaskTrigger -Daily -At 11:05PM
$settings = New-ScheduledTaskSettingsSet -StartWhenAvailable -DontStopIfGoingOnBatteries -AllowStartIfOnBatteries
Register-ScheduledTask -TaskName "XiRang-DailyFetch" -Action $action -Trigger $trigger1,$trigger2 -Settings $settings -Description "息壤每日数据拉取" -RunLevel Highest
```

每天 11:05 和 23:05 各跑一次，`-StartWhenAvailable` 确保错过的任务在开机后补跑。

## 从仿真到实盘

```
Phase 1: Paper Trading（当前）→ 3-6 个月验证
Phase 2: 人工执行（系统出指令，你手动买卖）→ 6-12 个月
Phase 3: 半自动（券商 API + 人工确认）→ 3-6 个月
Phase 4: 核心调仓全自动（出金永久人工）
```

切换只需改一个环境变量 `XIRANG_EXECUTOR`，核心引擎代码零修改。详见 [docs/09-实盘路线图.md](docs/09-实盘路线图.md)。
