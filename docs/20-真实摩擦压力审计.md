# 真实摩擦压力审计

> 本记录用于解释 `CAGR 7.60% / MDD -13.95%` 被真实世界摩擦压缩后的参考区间。
> 它不是税务意见，不是券商收益承诺，也不是实盘杠杆批准。

## 一、结论

当前 `90% 防御核心 + 10% 现代贝塔卫星` 的研究净值口径为：

- `CAGR 7.60%`
- `MDD -13.95%`

加入参数化真实摩擦后，当前审计参考结果为：

| 场景 | CAGR | MDD | 说明 |
|------|------|-----|------|
| `research_current` | `7.60%` | `-13.95%` | 当前研究口径，已含再平衡摩擦、动态压力滑点、50bp 恐慌卖出惩罚 |
| `unlevered_base_case` | `5.75%` | `-15.32%` | 无杠杆真实摩擦基础压力 |
| `unlevered_harsh_case` | `3.53%` | `-16.99%` | 无杠杆保守压力 |
| `levered_1_15_reference` | `5.90%` | `-17.94%` | 1.15x 融资压力参考，不是实盘批准 |

核心结论：

- `7.60% / -13.95%` 不能当作实盘可得净值。
- 更诚实的无杠杆真实摩擦参考区间，应先看 `5.75% / -15.32%`。
- 如果税务、预扣税和运行失败更重，净后年化可能下降到 `3.5%` 附近。
- 1.15x 在当前参数下并没有把净后收益显著拉高，反而把 MDD 推到接近 `-18%`。

## 二、模型假设

脚本：`backtest/real_world_friction_audit.py`

当前本地价格数据只有 adjusted close，没有逐笔分红、税务 lot、真实成交队列和账户利息流水。因此本审计采用参数化压力模型：

| 摩擦项 | base case | harsh case |
|--------|-----------|------------|
| 股息预扣税拖累 | `55bp / 年` | `75bp / 年` |
| 资本利得税或税务实现拖累 | `35bp / 年` | `75bp / 年` |
| 券商现金/融资利差拖累 | `10bp / 年` | `25bp / 年` |
| 运行失败折价 | `20bp / 年` | `50bp / 年` |
| 年度尾部失败冲击 | 每年最差日额外 `50bp` | 每年最差日额外 `150bp` |

`levered_1_15_reference` 使用：

- 杠杆：`1.15x`
- 融资利率：`4.79%`
- base case 的税务、预扣税、运行失败假设

IBKR 官方公开资料显示，美元现金利息上限约为 `3.12%`，且前 `10,000` 美元闲置现金不计息；美元保证金融资利率按余额分层，公开对比页给出的 `300K` 美元 margin loan 约为 `4.79%`。这些利差会直接侵蚀加杠杆后的净收益。

## 三、解释边界

这不是税务模拟器。

没有包含：

- 账户级税务居民身份差异。
- 税务 lot、wash sale、税损收割成功率。
- 真实股息现金流和逐笔预扣税。
- IBKR 每日实际现金利息入账。
- 真实 order book、成交队列和市场冲击。
- 券商在极端市场中临时提高保证金要求。
- 跨境清算、换汇、申报和合规成本。

这些只能在 Stage 9.5 之后，通过真实只读账户流水、券商报表和税务结构确认。

## 四、当前判断

`90/10` 生产候选仍然有研究价值和治理价值，但它的目标必须降温：

- 研究口径：`7.60% / -13.95%`
- 无杠杆真实摩擦参考口径：约 `5.75% / -15.32%`
- 保守压力口径：约 `3.53% / -16.99%`

如果最终目标是实盘长期可得 `7%+`，当前 90/10 候选还不够。它更像一个防御底座，而不是足以单独承担高收益目标的完整家办资产系统。

## 五、下一步

最合理的下一步不是上杠杆，而是：

1. 把 `backtest/real_world_friction_audit.py` 接入 PR 验证清单。
2. 用真实账户身份和税务结构替换当前参数假设。
3. 在 Stage 9.5 只读观察期内，用 IBKR 实际现金利息、保证金融资、报价和账户报表校准这些拖累参数。
