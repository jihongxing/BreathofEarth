# 息壤（Xi-Rang）

> 息壤：中国神话中能自己生长、永不耗尽的土壤。寓意财富的自我修复与持续增长。

## 一句话定义

一个状态机驱动的自动化资产配置系统，通过分层管理和风控优先机制，实现"稳定优先、抗通胀、防止人为犯错"的长期财富管理。

## 当前阶段

- Core 已完成最小真实交易能力骨架，但默认仍应关闭真实下单开关
- 系统当前优先级不是“扩大自动化”，而是“保持边界、观察运行、减少误操作”
- Alpha/Lab 继续与 Core/Stability 隔离，不进入正式收益汇报

## 核心理念

这个系统不是在"赚钱"，而是在"防止你做错事"。

- 优先级1：稳定（控制回撤）
- 优先级2：抗通胀（购买力不下降）
- 优先级3：收益（不以牺牲稳定为代价）

## 使用场景

- 自用 / 家族级别
- 封闭组织或联盟内部（成员制，非公开募集）
- 不面向公众，不触碰监管红线

## 与传统产品的区别

| 维度 | 息壤 | 传统基金/Robo-Advisor |
|------|------|----------------------|
| 核心目标 | 稳定优先 | 规模/收益优先 |
| 风险态度 | 主动控制 | 被动接受 |
| 用户行为 | 限制犯错 | 放任自由 |
| 系统形态 | 控制系统 | 金融产品 |

## 用户主动行为边界

- 入金可以自动化：资金先进入 Stability，再按规则补充 Core
- 出金不能自动化：系统只允许发起请求，后续必须人工审批、人工执行
- Alpha/Lab 不是家族主账户的一部分，必须与 Core/Stability 隔离运行
