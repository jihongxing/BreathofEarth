# 组织与合规备忘

> 本文档仅为思路记录，不构成法律建议。实际操作前必须咨询专业律师。

## 使用模式

### 模式一：纯自用 / 家族

- 不涉及外部资金，无合规问题
- 系统仅管理自己的账户
- 最简单，推荐先从这里开始

### 模式二：封闭组织 / 联盟

- 成员制，非公开募集
- 成员需签署协议，明确风险自负
- 资金走独立券商/银行账户，不与组织自有资金混用

## 关键注意事项

- 中国大陆：代客理财法律边界模糊，即使内部成员也需谨慎
- 美国：家族办公室有 SEC 注册要求（管理资产超过一定规模）
- 新加坡：家族办公室框架相对友好，但也有门槛
- 无论哪种模式，都建议资金托管在正规券商/银行，系统只负责生成指令

## 当前阶段结论

先做模式一（纯自用），用模拟资金验证系统。合规问题等系统验证通过、确定要扩展时再处理。
