# Benchmark 对比审计

> 本记录用于回答：`90% 防御核心 + 10% 现代贝塔卫星` 在知名公开基金/ETF 面前到底算什么水平。
> 对比必须同时展示研究口径和真实摩擦口径，不能只拿 `7.60%` 去比较。

## 一、命令

```bash
python -m backtest.benchmark_comparison_audit
```

脚本会：

- 使用 `90/10` 生产候选研究净值。
- 套用 `unlevered_base_case` 真实摩擦口径。
- 拉取或读取本地缓存的 benchmark adjusted close。
- 按每个 benchmark 的可用重叠区间重算候选系统表现。

`RPAR` 历史较短，不能和 2005 年起跑的全周期直接比较，所以脚本会自动把比较窗口切到 `2019-12-13` 之后。

## 三、解释

### 1. 对 `VBIAX / PRPFX`

真实摩擦口径下，候选系统收益不占优：

- vs `VBIAX`：真实摩擦 CAGR 低约 `2.23%`
- vs `PRPFX`：真实摩擦 CAGR 低约 `2.58%`

但候选系统的回撤明显更浅：

- vs `VBIAX`：MDD 浅约 `20.58%`
- vs `PRPFX`：MDD 浅约 `11.85%`

这说明它不是收益领先型系统，而是回撤治理型系统。

### 2. 对 `RPAR`

在 `RPAR` 可用历史窗口内，候选系统真实摩擦口径同时占优：

- CAGR 高约 `4.15%`
- MDD 浅约 `14.84%`

但这个窗口只有约 `6.4` 年，不能把它解释成跨 2008 全周期优势。

## 四、当前定位

这套 `90/10` 候选的正确商业定位是：

- 不是收益碾压 `VBIAX / PRPFX` 的进攻基金。
- 是一个明显降低深回撤的防御底座。
- 如果目标是税后、券商后、故障后仍达到 `7%+`，还需要新的收益来源，不能只靠当前 `90/10`。
- 如果目标是减少极端年份穿透风险，它有明确优势。
